📱 Telegram Setup

  1. Message @BotFather on Telegram → /newbot → copy token → set as TELEGRAM_BOT_TOKEN
  2. Message @userinfobot → copy your chat_id → set as TELEGRAM_CHAT_ID
  3. Register webhook (after Railway deploy):
    curl "https://api.telegram.org/bot{TOKEN}/setWebhook?url=https://YOUR-URL/telegram/webhook"
  4. Message your bot: /positions /pnl /account /close AAPL /closeall
